# Docker Python Server 🐳🐍

Простой веб-сервер на Python (Flask), который работает внутри Docker контейнера.

## Функции

- 🌐 Веб-интерфейс с информацией о контейнере
- 📡 REST API с информацией о времени, статусе и данных
- 💚 Health check для оркестрации
- 📊 Счетчик запросов
- 🔄 Echo API для тестирования
