# 🎙️ Транскрипция MP3 в TXT

CLI-утилита для преобразования аудиофайлов (MP3 и другие форматы) в текст с использованием локальной модели [OpenAI Whisper](https://github.com/openai/whisper).

## Возможности

- 🔒 **Полностью локально** — аудио не покидает ваш компьютер, модели загружаются один раз
- 📊 **Прогресс-бар** — посекундный прогресс транскрипции для длинных файлов
- 🌍 **Автоопределение языка** или явное указание (`ru`, `en`, `de` и др.)
- ⚡ **Гибкий выбор модели** — от быстрой `tiny` до точной `large`
- 🖥 **Поддержка CPU и GPU** — автоматически определяет CUDA при наличии

## Требования

| Зависимость | Минимальная версия |
|---|---|
| Python | 3.8+ |
| ffmpeg | любая |
| ОЗУ | ≥ 4 ГБ (для моделей tiny–small) |

### Установка ffmpeg

**macOS:**
```bash
brew install ffmpeg
```

**Ubuntu / Debian:**
```bash
sudo apt update && sudo apt install -y ffmpeg
```

## Установка

```bash
# 1. Клонируйте репозиторий (если ещё не)
# git clone <repo-url> && cd text

# 2. Установите Python-зависимости
python3 -m pip install -r requirements.txt
```

> При первом запуске модель Whisper будет загружена автоматически в кэш `~/.cache/whisper`.

## Использование

### Базовый

```bash
python3 transcribe.py input.mp3
# → создаст input.txt рядом с файлом
```

### Все опции

```bash
python3 transcribe.py input.mp3 \
  -o result.txt \          # выходной файл (по умолчанию: <имя>.txt)
  -m small \               # модель: tiny, base, small, medium, large
  -l ru \                  # язык аудио (автоопределение, если не указан)
  --progress               # показать прогресс в процентах
```

### Примеры

```bash
# Быстрая транскрипция с лёгкой моделью
python3 transcribe.py podcast.mp3 -m tiny

# Точная транскрипция русского аудио с прогрессом
python3 transcribe.py lecture.mp3 -m small -l ru --progress

# Явный вывод в указанный файл
python3 transcribe.py meeting.mp3 -o notes/meeting.txt
```

## Модели Whisper

| Модель | Параметры | Скорость (CPU) | Точность | ОЗУ |
|---|---|---|---|---|
| `tiny`   | 39 M  | ⚡⚡⚡ | ★★★ | ~1 ГБ |
| `base`   | 74 M  | ⚡⚡   | ★★★★ | ~1 ГБ |
| `small`  | 244 M | ⚡     | ★★★★★ | ~2 ГБ |
| `medium` | 769 M | 🐢     | ★★★★★ | ~5 ГБ |
| `large`  | 1550 M| 🐌     | ★★★★★+| ~10 ГБ |

> 💡 **Рекомендация:** начните с `small` — хороший баланс скорости и точности. Используйте `--progress` для файлов длиннее 5 минут.

## Режим прогресса

Флаг `--progress` разбивает аудио на чанки (по умолчанию 30 с) и показывает процент выполнения:

```bash
python3 transcribe.py long_podcast.mp3 --progress --chunk-seconds 60
```

## Устранение неполадок

| Проблема | Решение |
|---|---|
| `ffmpeg не найден` | Убедитесь, что `ffmpeg` в PATH: `which ffmpeg` |
| `Пакет 'whisper' не найден` | `pip install -r requirements.txt` |
| Медленная транскрипция | Используйте модель `tiny` или `base`, либо GPU с CUDA |
| Ошибка загрузки модели | Проверьте интернет-соединение — модель скачивается при первом запуске |
